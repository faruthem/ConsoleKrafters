primeramente es necesario ejecutar el siguiente comando para formar o crear el cliente

localizar tu dirección en terminal una dirección arriba de la actual con el comando "cd.."

paso 2: ejecutar el siguiente comando de next.js para armar el cliente

    npx create-next-app@latest krafters_frontend
paso 3: realizar los siguientes opciones:
        TypeScript No 
        ESLint NO 
        Tailwind CSS Si (en mi caso)
        src/ directory Yes
        App Router Yes
        import alias Yes y agregamos @/*
paso 4: para comprobar que funciona nos metemos dentro de la carpeta del proyecto y hacemos un yarn dev
